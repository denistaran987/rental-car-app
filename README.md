### Installation & Usage

1. **Clone the repository**

```bash
git clone https://github.com/your-username/car-rental-app.git
cd car-rental-app
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the development server**

```bash
npm run dev
```

4. **Build for production**

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
